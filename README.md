Takealot Stock Reconciliation & Claim Tool Overview:

This is a professional tool designed to help Takealot sellers analyze stock reconciliation reports and recover lost revenue by identifying claimable discrepancies and generating submission-ready invoice

Problem:

Takealot sellers receive CSV reconciliation reports that are:

Hard to interpret Time-consuming to process Easy to miss claimable stock

Solution:

This application automates the full workflow:

Upload reconciliation CSV Detect missing/claimable stock Generate professional claim invoice Prepare submission to Takealot support

Features: 📊 CSV Processing: Handles Takealot CSV format Ignores description rows Dynamically detects columns 🧾 Invoice Generation Claim-specific invoice format Seller account identification Clean, professional layout 📁 Bulk Scanner: Upload multiple CSV files Detect claimable reports instantly Aggregate total claim values Account-level grouping 📤 Submission Assistant: Auto-generated support message Direct Takealot support link Required document checklist 🛠 Tech Stack: React (Vite) Tailwind CSS PapaParse (CSV parsing) Lucide Icons

☁️ Cloud Architecture (Planned): File storage (AWS S3) Serverless processing (AWS Lambda) API layer (API Gateway) Database (DynamoDB/PostgreSQL) Authentication (Cognito)

📊 Real Impact: Helps sellers quickly identify and recover thousands of rands in lost stock.
